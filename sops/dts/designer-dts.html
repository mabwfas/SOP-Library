Morning Block (9:00 AM - 12:00 PM)

  • Check design requests - Review new tasks from PM
  • Client feedback review - Address revision requests first
  • Brand research - Study client brand for new projects
  • Create wireframes - Start with low-fidelity layouts
  • Organize Figma files - Keep projects structured

Afternoon Block (12:00 PM - 4:00 PM)

  • High-fidelity designs - Create polished mockups
  • Mobile responsiveness - Design mobile versions
  • Component creation - Build reusable design components
  • Design review prep - Prepare presentations for clients
  • Asset export - Export icons, images for developers

Evening Block (4:00 PM - 6:00 PM)

  • Submit designs - Send to PM for review
  • Developer handoff - Prepare specs and assets
  • EOD status update - Report progress
  • Design inspiration - Browse Dribbble, Behance for trends

Pro Tip

Mobile first, always. 70% of ecommerce traffic is mobile. Design for mobile first, then scale up to desktop.
